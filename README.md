## 📖 Project Story / История на проекта
In today’s fast-paced educational environment, efficiently managing students, courses, and academic operations is a challenge. Inspired by platforms like **Moodle**, this project was created to **combine software development skills with educational management concepts**.  

The goal was to build a **simplified, yet functional Moodle-like system** using **C++ and Object-Oriented Programming (OOP)**. The project demonstrates how **modular design, class inheritance, and encapsulation** can be applied to solve real-world problems, while also highlighting skills in **algorithm implementation, data structures, and software design patterns**.  

This project tells the story of **turning academic theory into practical solutions**: managing users, courses, and student enrollments through a **console-based application**, creating a foundation for **scalable, maintainable educational software**. It emphasizes **problem-solving, logical thinking, and clean coding practices**, which are highly valued in any technical role.  

By working on this project, I gained hands-on experience in **C++ programming**, **OOP concepts**, **console application development**, and **algorithmic thinking**, while creating a system that simulates **real-world educational workflows**.

### 🛠️ Methodology / Процес
- **System Design:** Defined core classes (Student, User, Course, Admin) and their relationships  
- **Class Implementation:** Built constructors, methods, inheritance, and encapsulation  
- **Operations Handling:** Created functions to add/remove users, enroll students, and manage courses  
- **Console Interaction:** Interactive menu system for testing and usage  
- **Testing & Validation:** Verified each module for correctness and handled edge cases  
- **Modularity:** Structured code for easy extension (grades, notifications, additional features)
